## 1. Abstract (摘要)

**English:**
This document outlines the **Rosetta Protocol**, a standard for compressing human language into the 44-bit Payload defined in SCA. It also establishes the **Global Rosetta Map**, a decentralized, CC0-licensed universal semantic ID dictionary co-constructed by humanity.

**中文:**
本文档概述了 **罗塞塔协议 (Rosetta Protocol)**，这是一个将人类语言压缩进 SCA 定义的 44-bit Payload 的标准。同时，本文确立了 **Global Rosetta Map**，一个全人类共建的、去中心化的、CC0 许可的通用语义 ID 字典。

## 2. The Rosetta Protocol (罗塞塔协议)

**Objective**: To define standard algorithms (VSCP) for compressing human language into the 44-bit Payload.
**目标**: 定义将人类语言压缩进 44-bit Payload 的标准算法 (VSCP)。

### 2.1 Compression Logic (压缩逻辑)

*   **Smart Abbreviation (智能缩写)**:
    *   Algorithmic reduction of common terms.
    *   Example: `transaction` -> `trx7n`
*   **Glyph Indexing (字模索引)**:
    *   Mapping distinct characters/glyphs to compact IDs.
    *   Example: `已发货` (Shipped) -> `[12][12][12]`

## 3. The Global Rosetta Map (全球罗塞塔映射图)

### 3.1 Standard First (标准先行)

**Project LOGOS** operates not just as a software project, but as an **Open Source Standard Committee**.

**Core Assets (核心资产)**:

1.  **Global Rosetta Map**:
    *   A universal Semantic ID table built by all of humanity.
    *   **License**: CC0 (No Copyright Reserved).
    *   **Vision**: Ensuring that `ID#Admin` means "Admin" on every Logos-compatible machine in the universe.
    *   **全人类共建的通用语义 ID 表 (CC0 协议)**。

2.  **The Negentropy Manifesto (计算熵减宣言)**:
    *   A consensus document uniting global developers against meaningless complexity.
    *   **凝聚全球开发者的共识文件**。

### 3.2 Compute-View Separation via H-PU (基于 H-PU 的算视分离)

The Map enables the **H-PU (Human Presentation Unit)** to function.

*   The Machine processes the **ID**.
*   The H-PU queries the **Map** to display the **Meaning**.

This separation allows detailed, human-readable semantics without incurring any runtime or bandwidth cost during calculation.
